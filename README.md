# React Native Movie App

This is a movie app made with react native for learning purposes.

## Run

1. Install react native and configure environment

2. Run the bundler
```
$ npm start
``` 
2. Run the bundler
```
$ npm start
``` 
Or with yarn
```
$ yarn start
``` 
3. Run in your emulator or device
```
npm run android
```
Or
```
npm run ios
```
